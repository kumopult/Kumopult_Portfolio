### VTuber——神乐阿库娅
这个模型中尝试了很多入门shader后才学到的技术，比如用顶点色控制描边颜色和宽度、用类似本村线的手法标记固定阴影区域，还把UV展成方格状防止纹理走样、甚至为了配合衣服上的斜方格把UV块给倾斜了45度。

效果有好有坏，但总归是尝试。
