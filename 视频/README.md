# 视频作品
投手作为up主在b站的一些代表作

早期做的主要是一些自制DNF补丁的展示视频，从专业角度看其实没啥技术含量。
之后逐渐学了AE，开始做一些飙特效的短视频，比如宝具动画。
再之后是接触了VTuber，做了一些歌曲PV，算是达到了入门的后期水平。

3D动画则是在20年接触blender后才开始做的，不过那之后心态和以前有了些变化，从不择手段地出成果变成了先追求技术力，结果一年都没做出啥新的视频作品，转而去做游戏和插件了……
